Dime

Dime is an iOS application designed to simplify sending expense reports. I wrote it in Swift over the summer of 2015 after deciding that I wanted to learn how to create mobile applications.

It aims to make expense reports easy without requiring a third-party web account to compile the expenses into a report.

Physio

Physio is an application that acts as an architecture for communication between a Microsoft Band smart watch, Android smart phone and an ACT-R/Φ cognitive architecture.

I created it in the summer of 2016 while doing research at Bucknell University.

Pokemon Orange & Blue

Pokémon Orange & Blue is a version of the Pokémon games written in Java and created by myself and 3 others in the spring of 2016 for a Software Engineering and Design class. Our goal was to create a game that felt like one of the original Pokémon games. I was in charge of the graphical interface and created all of the menus and visual interactions.
