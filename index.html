Short.ly

Shortly is a simple URL shortener inspired by some of my interview questions at HubSpot. The server exposes an API for each shortened URL that posts an event feed using Jersey's Server Sent Events.

The system is composed of a React frontend (served by nginx), and a Dropwizard/MongoDB backend. All modules in the project are Dockerized, and built with pmake.

pmake

A build tool, that allows for specification of multiple tasks and task groups. All tasks in a group are run simultaneously, allowing for better build times, and guaranteeing that all prerequisite tasks are completed before the next group starts.

Custom output can be specified for failure and success messages using the YAML formatted pmake file.

GitHub

Dime

Dime is an iOS application designed to simplify sending expense reports. Its main goal is to make expense reports easy by not requiring a third-party web account to compile or send a group of expenses as a report.

Dime compiles recorded expenses into PDFs, generating them on the device itself, meaning no internet is required to push a report to the user's outbox.
